# vue-start

> A Vue.js project

剖析`vue`原理，暂时没有考虑健壮性

实现了
 * `vue-router`
 * `vuex`
 * `MVVM框架`

插件代码都放在src/components下面

# 运行方法
1. 克隆代码：

```
git clone https://github.com/eyes487/vue-start.git
```

2. 进入项目根目录并安装依赖：

```
cd vue-start
npm install /yarn
```

3. 运行项目

```
npm start  /yarn start

在浏览器中输入http://localhost:8080

```
